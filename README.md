# FAWKES — Intercept-Sim v04.11

Ballistic intercept validation simulator with 3D altitude axis and gravity physics.

---

## Running Locally

**Prerequisites:** [Node.js](https://nodejs.org/) (v18 or higher) must be installed.

### First-time setup

Install dependencies (only needed once):

```bash
npm install
```

### Start the dev server

```bash
npm run dev
```

Then open your browser and navigate to:

```
http://localhost:5173/
```

To stop the server, press `Ctrl + C` in the terminal.

---

## Building for Production

```bash
npm run build
```

Output will be in the `dist/` folder.

---

## Tech Stack

- **Vite** — build tool & dev server
- **Vanilla JS** — no framework
- **HTML5 Canvas** — 3D isometric rendering
