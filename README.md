# FAWKES

## Running Locally

**Prerequisites:** [Node.js](https://nodejs.org/) must be installed.

### First-time setup

Install dependencies (only needed once):

```
npm install
```

### Start the dev server

```
npm run dev
```

Then open your browser and navigate to:

```
http://localhost:5173/
```

To stop the server, press `Ctrl + C` in the terminal.
